##  아레나 진행 방식
기본모드:  30분 (배틀 파트) -> 5분 휴식 (상점파트) -> 30분 (배틀파트) -> 5분 휴식 (상점파트)... 

배틀파트: 적 무한 랜덤 리스폰, 골드 드랍 

즉 성장 컨텐츠 = 상점 활용 가능한 "골드" 재화 (경험치가 아님)

(일단 지금은 기본 모드만 구현!!)

## 상점
좌측에 상점 이미지 스프라이트, 우측에 품목 

품목은 스테이지가 지나도 고정, 고급 아이템들은 가격을 높게 설정해서 어차피 초반 라운드엔  못 사게

품목 구매 수량은 무제한

상점: 8x3으로 슬롯 만들기

## UI 
좌측 하단: 캐릭터 스프라이트 and 체력 게이지 and 무기 선택 and 표정 변화
무기 선택: 1,2,3,4 버튼으로 바뀌게, 현재 장착 중인 무기만 표시되게.
표정 변화: < > 키로 좌우 변경 시스템

중앙: 마인크래프트식 인벤토리 UI 

배틀 파드 클리어 시 canvas group으로 "클리어!" 등장 후 사라짐


I 키 = 인벤토리 팝업
